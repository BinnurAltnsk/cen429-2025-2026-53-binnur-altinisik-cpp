Detailed Description

TLS bağlamı (context) yapısı

Bu yapı, TLS bağlantı bilgilerini saklar. Gerçek implementasyonda OpenSSL SSL_CTX veya benzeri kullanılabilir.

Note
Bu yapı, TLS bağlantısının yaşam döngüsü boyunca geçerli kalır. Bağlantı kapatıldığında cleanupTLSContext() ile temizlenmelidir.
