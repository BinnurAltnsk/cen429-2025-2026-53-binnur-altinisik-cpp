ErrorCode TravelExpense::Database::closeDatabase ( sqlite3 *  db)

Veritabanı bağlantısını kapat.

Açık olan SQLite veritabanı bağlantısını kapatır ve kaynakları temizler. Tüm prepared statement'lar ve transaction'lar kapatılır.

Note
Bu fonksiyon, nullptr handle'ı geçerli kabul eder ve Success döner. Bağlantı kapatıldıktan sonra, handle geçersiz hale gelir.
Parameters
dbVeritabanı handle'ı (nullptr ise hiçbir işlem yapılmaz)
Returns
ErrorCode Başarı durumu (Success, FileIO vb.)

Veritabanı tablolarını oluştur (migration)

Veritabanı handle'ını al (singleton pattern)

Mevcut veritabanı bağlantısını döndürür. Eğer bağlantı yoksa yeni bir bağlantı oluşturur (initializeDatabase() çağrılır).

int64_t TravelExpense::Database::getLastInsertRowId ( sqlite3 *  db)

Son eklenen satırın ID'sini al.

Son INSERT işleminde eklenen satırın auto-increment ID'sini döndürür. SQLite'in last_insert_rowid() fonksiyonunu kullanır.

◆ initializeDatabase()

sqlite3 * TravelExpense::Database::initializeDatabase ( const char *  dbPath = nullptr)

Veritabanı bağlantısını başlat.

SQLite veritabanı dosyasına bağlantı açar. Dosya yoksa oluşturur. Veritabanı dosyası, belirtilen yolda oluşturulur veya açılır.

Note
Bu fonksiyon, veritabanı dosyasının bulunduğu dizini oluşturmaz. Dizin önceden oluşturulmuş olmalıdır. Eğer dizin yoksa, hata döner.
Parameters
dbPathVeritabanı dosya yolu (nullptr ise "data/travelexpense.db" kullanılır) Dosya yolu geçerli olmalı ve dizin mevcut olmalı.
Returns
sqlite3* Veritabanı handle'ı, nullptr ise hata (dosya açılamadı, izin hatası vb.)

ErrorCode TravelExpense::Database::resetDatabase ( )

Veritabanı bağlantısını sıfırla (yeniden bağlan)

Mevcut bağlantıyı kapatır ve yeni bir bağlantı açar. Singleton pattern'deki bağlantıyı yeniden başlatır.

Note
