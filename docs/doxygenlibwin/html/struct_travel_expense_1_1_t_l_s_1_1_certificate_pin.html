Detailed Description

Sertifika pin bilgisi.

Bu yapı, pin'lenecek sertifika bilgilerini saklar. Certificate pinning, sunucu sertifikasının beklenen değerlerle eşleşmesini zorunlu kılar.

Note
Hem fingerprint hem de public key hash pin'lenebilir. En güvenli yaklaşım, her ikisini de pin'lemektir.
