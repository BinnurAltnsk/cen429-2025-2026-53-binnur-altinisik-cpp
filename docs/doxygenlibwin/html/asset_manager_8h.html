Detailed Description

Seyahat Gideri Takibi - Varlık Yönetimi.

Bu dosya, Varlık Yönetimi gereksinimini karşılar. Her varlık için aşağıdaki özellikler belirlenmeli ve dokümante edilmelidir:

  • Adı
  • Açıklaması
  • Konumu (ör. veritabanı, tablo, sütun)
  • Kaynağı
  • Boyutu
  • Oluşturulma Zamanı
  • Silinme Zamanı
  • Varsayılan Değeri
  • Varlık Koruma Şeması (Gizlilik, Bütünlük, Kimlik Doğrulama)
