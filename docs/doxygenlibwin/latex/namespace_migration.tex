\doxysubsection{Detailed Description}
Binary → SQLite migration işlemleri modülü 

Bu modül, binary dosya tabanlı veri saklama sisteminden SQLite veritabanına geçiş işlemlerini yönetir. Veri migrasyonu, veri doğrulama ve rollback işlemlerini sağlar.
