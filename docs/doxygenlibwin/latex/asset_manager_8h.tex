\item Adı
\item Açıklaması
\item Konumu (ör. veritabanı, tablo, sütun)
\item Kaynağı
\item Boyutu
\item Oluşturulma Zamanı
\item Silinme Zamanı
\item Varsayılan Değeri

\end{DoxyItemize}
