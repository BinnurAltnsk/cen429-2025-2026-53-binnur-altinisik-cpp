\doxysubsection{Detailed Description}
Güvenli string işlemleri modülü 

Bu modül, platform-\/bağımsız güvenli string işlemleri sağlar. MSVC\textquotesingle{}de strncpy\+\_\+s gibi güvenli fonksiyonlar kullanır, diğer platformlarda standart fonksiyonlar + null terminator kontrolü yapar. MSVC C4996 uyarılarını önler.
